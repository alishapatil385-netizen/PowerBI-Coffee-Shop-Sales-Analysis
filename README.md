# PowerBI-Coffee-Shop-Sales-Analysis

**📌 Project Overview**

The Coffee Shop Sales Dashboard is an interactive Power BI project designed to analyze sales performance, customer purchasing patterns, and product category trends. The dashboard transforms raw sales data into meaningful insights that help understand business performance and support data-driven decision-making.

**🎯 Objectives**
Monitor overall sales performance.
Track monthly revenue trends.
Analyze transaction volume and average order value.
Identify top-performing product categories.
Discover customer purchasing patterns.
Provide actionable business insights through visualization.

**📊 Dashboard Features**
1. KPI Cards
Total Revenue
Total Transactions
Average Order Value
2. Revenue Trend Analysis
Monthly revenue performance
Sales growth and decline identification
Seasonal trend monitoring
3. Product Category Analysis
Revenue contribution by category
Best-selling product groups
Category-wise performance comparison
4. Interactive Filters
Product Category
Date Selection
Dynamic Dashboard Navigation

**🛠️ Tools & Technologies**
Microsoft Power BI
Microsoft Excel
DAX (Data Analysis Expressions)
Data Visualization Techniques

**📂 Dataset Information**
The dataset contains coffee shop transaction records including:
Transaction Date
Product Category
Product Type
Quantity Sold
Unit Price
Revenue

**📈 Key Insights**
Revenue showed a consistent upward trend during the observed period.
A small number of product categories contributed the majority of total sales.
Average order value remained relatively stable across transactions.
Certain months generated significantly higher revenue, indicating seasonal demand patterns.
